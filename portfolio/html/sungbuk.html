프로젝트 개요

성북구에서 운영하는 공공시설의 관리나 사업관리를 위한 플랫폼

담당업무

Java를 이용한 프로젝트 백엔드의 전반을 담당하였으며, docker로 서버를 구축하고 git으로 버전관리를 하였습니다.

스프링시큐리티를 이용하여 로그인기능을 구현하였습니다.

MariaDB 를 이용하여 데이터 관리하였으며, 좌표를 저장하며 범위를 지정하여 범위안의 좌표를 셀수있는 프로시저를 작성하였습니다.

Json 형식으로 이력을 관리하면서 저장, 임시저장에 따른 데이터 표시 변경등 전체적인 CRUD를 담당하였습니다.
